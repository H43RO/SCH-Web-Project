사상 초유의 팬데믹 사태.
그동안 우린 고통속에 살아왔습니다. 나날이 심각해지는 코로나19 사태는 진정될 줄을 모르고,
잠잠해질만 하면 다시 터지고, 또 터져 끝나지 않는 악순환이었습니다.
하지만 국민들의 단합으로 이제는 점차 나아지고 있는 상황입니다.
사회적 거리두기만 실천한다면, 마음 한 켠에 묵혀놨던 여행 계획을 꺼내도 좋습니다.
개발자 소개
해당 사이트는 순천향대학교의 3학년 과정 '고급 웹 프로그래밍' 과목의 Term Project로 제작되었습니다.
HTML, CSS, JavaScript 등을 사용했습니다. WAS 구현이 없기 때문에 로그인, 공지사항 등의 기능은 정적으로 동작합니다.


김현준 (H43RO)

순천향대학교 사물인터넷학과 3학년을 재학 중입니다.
보유한 기술 스택은 다음과 같습니다.
  • C, C++ 을 활용한 임베디드 시스템 개발
  • Java, Kotlin 을 활용한 안드로이드 앱 개발
  • Python 을 활용한 백 엔드 개발 (Django)
이 외에도 HTML/CSS/JS, Swift 를 활용한 iOS 앱 개발 등을 간단하게 할 줄 압니다.
주력 기술은 안드로이드 앱 개발이며, 다양한 외주 및 사이드 프로젝트를 진행한 바 있으며
개인적으로 2개의 앱을 런칭한 경험이 있습니다. ('동백고 모바일', '나는 술찌다' 등)

RxKotlin, RxAndroid 를 활용한 Reactive Programming 에 관심을 보이고 있으며,
JetPack Component 을 활용하여 트렌디하게 앱을 개발하는 것을 목표로 합니다.
